# AI-Powered LinkedIn Content Creator Automation

This project is an AI automation workflow built using n8n that generates LinkedIn posts automatically.

## Features
- Fetch content topics from Google Sheets
- Generate posts using AI (Google Gemini)
- API integrations for content enhancement
- Store output back into database

## Tech Stack
- n8n
- Google Sheets
- Gemini API
- HTTP Requests

## Workflow Steps
1. Read topic from sheet
2. Process using AI
3. Save generated post
